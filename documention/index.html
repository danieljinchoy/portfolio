Customization

We don’t suggest you edit the original css and js files. Because the template update will cover the original files. You could add a new css or js file. Add the customized codes into the files.
You could find the custom.css in the css folder, we suggest you clone it as your own CSS file. Edit the new CSS file.

HTML Structure

Introduction

All of the information within the main content area is nested within a div with an id of "wrap-all". Here is the general structure.
All style.css are put before </head>, put all js files(In addition to jquery.js) before </body>. 

<div id="wrap-all">
    <!-- Page Loading -->
    <div class="page-loading fullscreen-wrap visible"...>...</div>
    <!-- Audio Player -->
    <div id="jquery_jplayer" class="jp-jplayer"...>...</div>
    <div id="wrap-outer">
        <!--Header-->
        <header...>...</header>
        <!-- Navigation for menu hide layout -->
        <div id="menu-panel"...>...</div>
        <div  id="portfolio-panel" class="portfolio-panel">...
        </div> <!--End End #portfolio-panel-->
        <!--#main-wrap-->
        <div id="wrap">...
        </div> <!--End #wrap-->
        <!--Popup for Header Icon on Header-->
        
    </div><!--End #wrap-outer-->
    <!-- Lightbox -->
    <div class="pswp" tabindex="-1" role="dialog" aria-hidden="true"...>...</div>
    <footer...>...</footer>
</div><!--End #wrap-all-->

Logo and Menu Color rule

For the dark or alt logo color, the class name should be:

Default Alternative
To show which Logo color on header default-logo alt-logo
To show which Menu color on header default-menu alt-menu
To show which Logo/Menu color on the menu panel default-logo-menu-expand alt-logo-menu-expand
To show which Logo color/Menu(header) for mobile default-logo-menu-mobile alt-logo-menu-mobile

If you want to show default color logo, please add class name default-logo into body node, otherwise, add class name alt-logo instead of default-logo.
You should add the alt image logo(learnmore), The text logo and menu default and alternative color value is in the custom.css(search "Logo text color").

Page from top

The content area will start from the page top edge instead of the header bar if the class name page_from_top is added to the body node. It is suitable to a page with hero section.

The class name non_bg_header(on body node) will force remove the background color from the header bar.

Lightbox Skin

The lightbox skin should be defined by the class name pswp-light-skin.

Header Sticky
Always Sticky

The header could be stayed on the top of the windown when the page is scrolling to the bottom.
Please add class name header-sticky to the body node.

Sticky when scrolling back

The header could be stayed on the top of the windown when the page is scrolling to top.
Please add class name header-sticky-back to the body node.

Please don't forget to remove the class name header-sticky-none from the body node.
